# Yet Another Pico Tetris

A small, polished, and technically compliant Tetris implementation built for [PICO-8](https://www.lexaloffle.com/pico-8.php).

This project focuses on providing a high-fidelity experience, featuring an SRS (Super Rotation System) implementation from scratch and a wide variety of challenge modes.

## Controls
* **Left/Right Arrow:** Move pieces horizontally
* **Down Arrow:** Soft drop
* **Up Arrow:** Hard drop
* **Z/X:** Rotate (Clock-wise / Counter clock-wise)
* **Z + X:** Hold piece

## Features
* **SRS Compliant:** Fully implemented Super Rotation System.
* **Advanced Mechanics:** Includes T-Spins and Mini-T-Spins for extra points.
* **Challenge Modes:** Each mode tracks unique Personal Best scores and times.

### Game Modes
* **Casual:** Clear 15 lines (no time pressure).
* **Marathon:** Clear 150 lines.
* **Quickie:** Clear Casual requirements in 3 minutes.
* **Reverse:** Use 40 pieces without clearing a line.
* **Expert:** Reach Level 10 (No Hold, No Ghost, 1-piece preview).
* **Heavy G:** Survive 2 minutes with increased gravity.
* **Perfect:** Achieve a Perfect Clear.
* **Tight:** Play on a 6-column board.
* **Garbage:** Clear the bottom row with rising garbage blocks.
* **Rush:** Clear lines to gain time and survive as long as possible.

## Technical Details
* **Platform:** PICO-8
* **Constraint:** Built under a very tight token limit (99.8% usage).
* **Source:** Written in Lua.
